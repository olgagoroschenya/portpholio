1.Создаю в Postman новую коллекцию  NEW Agentapp.
2.Аутентификация с помощью запроса POST, получаю token в ответе, с помощью сниппета Set an environment variable заношу token в Еnvironment (также в коллекции в Authorization выбрала Auth Type - API Key, key - token, Value - значение полученное в результате запроса, добавляю в Header).
3.Создание водителя с помощью запроса POST, в body запроса указываю данные водителя(first_namе, last_name, patronymic), в ответе получаю id водителя, с помощью сниппета Set an environment variable в Scripts заношу id водителя(id_driver) в Еnvironment.
4.Создание собственника с помощью запроса POST, в body запроса указываю данные собственника (тот же самый, что и водитель), в ответе получаю id собственника, с помощью сниппета Set an environment variable в Scripts заношу id собственника(id_owner) в Еnvironment.
5.Создание страхователя с помощью запроса POST, в body запроса указываю данные  (тот же самый, что и собственник), в ответе получаю id страхователя, с помощью сниппета Set an environment variable в Scripts заношу id страхователя (id_insurant) в Еnvironment.
6.Cоздание автомобиля с помощью запроса POST, в body запроса указываю данные автомобиля, в ответе получаю id автомобиля, с помощью сниппета Set an environment variable в Scripts заношу id автомобиля (id_car) в Еnvironment.
7.Cоздание объекта страхования с помощью запроса POST, в body запроса указываю данные водителя, собственника, автомобиля, страхователя(значениями являются переменные из Еnvironment), в ответе получаю id объекта страхования, с помощью сниппета Set an environment variable в Scripts заношу id объекта страхования (id_insured_object) в Еnvironment.
8. Cоздание договора с помощью запроса POST, в body запроса указываю необходимые парамерты для создания договора, в ответе получаю id договора, с помощью сниппета Set an environment variable в Scripts заношу id договора (agreement_id) в Еnvironment.

Договор со статусом "Одобрен" появляется в списке "Мои Договоры".
10.Обновление договора с помощью запроса PATCH, в body запроса указываю insured_object - значение является пременная id_insured_object из Еnvironment.
11.Получение полного расчета с помощью запроса POST, в консоль Postman вывожу "integrated_company_title" -- наименование Страховой Компании
"integrated_company_code" -- code Страховой Компании
"premium" -- Премия, 
для этого в Scripts делаю парсинг из body ответа.
